# filebrowser (default)

## What you get

- **Try locally**: 启动后访问 `http://localhost:8080` 打开 UI。
- **No external keys**: 基础体验不需要第三方 API key。

## Run

在本目录执行：

```bash
docker compose up -d
```

打开 `http://localhost:8080`。默认账号通常为 `admin/admin`（以项目文档为准），首次进入请修改密码。

## Stop

```bash
docker compose down -v
```
