# syncthing/syncthing (default)

## What you get

- **Try locally**: After startup, open `http://localhost:8385` in your browser.
- **No external keys**: No third-party API keys required.

## Run

In this directory:

```bash
docker compose up -d
```

Open `http://localhost:8385` and follow the onboarding prompts.

## Stop

```bash
docker compose down -v
```
