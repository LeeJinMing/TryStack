# Stirling-PDF

Self-hosted web app for PDF manipulation (merge, split, convert, and more).

## Quick start

```bash
npx --yes -p github:LeeJinMing/TryStack#v0.0.1 trystack up Stirling-Tools/Stirling-PDF
```

Open:

- `http://localhost:8109`

## Notes

- This is intended for local evaluation (Try locally).
