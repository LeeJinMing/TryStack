# qBittorrent (default)

## What you get

- **Try locally**: qBittorrent Web UI at `http://localhost:8080`.
- **Local storage**: Separate volumes for config and downloads.

## Run

In this directory:

```bash
docker compose up -d
```

Then open `http://localhost:8080` in your browser.

## Stop

```bash
docker compose down -v
```
