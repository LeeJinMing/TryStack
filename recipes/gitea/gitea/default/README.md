# gitea/gitea (default)

## What you get

- **Try locally**: After startup, open `http://localhost:3003` in your browser.
- **No external keys**: No third-party API keys required.

## Run

In this directory:

```bash
docker compose up -d
```

Open `http://localhost:3003` and follow the installation wizard (defaults are fine for local use).

## Stop

```bash
docker compose down -v
```
