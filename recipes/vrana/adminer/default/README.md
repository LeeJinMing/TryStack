# Adminer

Lightweight database management UI (works with MySQL, PostgreSQL, SQLite, and more).

## Quick start

```bash
npx --yes -p github:LeeJinMing/TryStack#v0.0.1 trystack up vrana/adminer
```

Open:

- `http://localhost:8101`

## Notes

- This recipe does not include a database. Connect it to your own database if needed.
